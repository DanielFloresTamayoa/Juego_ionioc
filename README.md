# Juego_ionioc
## Funcionalidad

El juego "Adivinar número", consiste en que nosotros acertemos un número aleatorio del 1 al 10, ingresando otro número desde un input.

**Versiones:**
  - node: v19.1.0
  - npm: 8.19.3
  - ng: angular/cli 15.0.1
  - cordova: 11.0.0
  - ionic: 6.20.4
